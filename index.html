Current Projects

SocietyWeb: Team Management App

SocietyWeb is a Team Management app designed for groups like Girl Scouts or Sports Teams. The application's frontend was built using React and styled with vanilla CSS utilizing flexbox. The backend of the application runs on a Node.js server using the Express framework and storing data with PostgreSQL. Twilio SMS and SendGrid APIs were also used to add team communication.

Technologies Used:

React, HTML, CSS, Node, Express, PostgreSQL, Mocha/Chai/Jest

One Love Rewards: Dining Rewards Platform

One Love Rewards is an open source restaurant rewards platform that allows business owners to implement a free to use program. The application's frontend was built using React and styled with vanilla CSS utilizing Grid. The backend of the application runs on a Node.js server using the Express framework and storing data with PostgreSQL. This application taught me the best practices of building a RESTful API.

Technologies Used:

React, HTML, CSS, Node, Express, PostgreSQL, Mocha/Chai/Jest
